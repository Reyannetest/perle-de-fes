## Animations Requises

1. **Ouverture page** : fade in 1.5s
2. **Titre hero** : slide in depuis la gauche
3. **Sections au scroll** : fade in + slide up (Intersection Observer)
4. **Photos galerie** : apparition cascade (délai 0.2s)
5. **Survol carte** : zoom scale(1.03) + overlay doré
6. **Clic produit** : modale avec scale + fade (0.4s)
7. **Hover boutons** : brightness(1.15)
8. **Navigation** : soulignement doré animé
9. **WhatsApp fixe** : rebond toutes les 5s
10. **Confirmation formulaire** : fade in avec icône ✓
