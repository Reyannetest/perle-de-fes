# Images — Perle de Fès

## Images à remplacer

Remplacez les images placeholder par vos vraies photos.
Glissez-déposez vos images dans ce dossier avec les noms exacts ci-dessous.

### Liste des images requises

| Fichier | Description | Dimensions recommandées |
|---------|-------------|------------------------|
| `hero-bg.jpg` | Photo principale du hero (ambiance hammam/riad) | 1920 x 900 px minimum |
| `panier-mini.jpg` | Panier Mini Hammam | 600 x 500 px |
| `panier-essentiel.jpg` | Panier Essentiel Hammam | 600 x 500 px |
| `panier-premium.jpg` | Panier Premium Hammam | 600 x 500 px |
| `dragees.jpg` | Dragées Marocaines Orientales | 600 x 500 px |
| `papeterie.jpg` | Papeterie Personnalisée | 600 x 500 px |

### Recommandations

- **Format** : JPG optimisé pour le web
- **Poids** : Maximum 300 Ko par image pour la performance
- **Qualité** : 80-85% de compression JPG
- **Ratio** : Les images produits doivent être au ratio 6:5 (600x500)
- **Style** : Fond neutre (beige/blanc cassé), éclairage doux, ambiance luxueuse

### Outils d'optimisation gratuits

- [Squoosh](https://squoosh.app) — Compression en ligne par Google
- [TinyPNG](https://tinypng.com) — Compression automatique
- [ImageOptim](https://imageoptim.com) — App Mac gratuite

### Après remplacement

1. Vérifiez que les noms de fichiers correspondent exactement
2. Testez le site en local (ouvrez index.html)
3. Committez et poussez vers GitHub :
   ```bash
   git add images/
   git commit -m "Ajout des vraies photos produits"
   git push
   ```
